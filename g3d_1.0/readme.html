Welcome to G3D

G3D is a modular graphics engine built on top of Marmalade and OpenGL ES 2.0. It is aimed at a similar level to XNA, or can be seen as a replacement for IwGx utilising OpenGL ES 2.0 rather than OpenGL ES 1.x style of data.

The documentation is here: docs/index.html.

This is the folder structure: 
g3d
  bin
    converter     This contains the converter to G3D format data.
  docs            This contains the html documentation
  samples         This contains example code.
  src
    g3d           This is the source for G3D.
    G3D.Converter This is the source for the converter

If you have issues with G3D, please contact me via GitHub. Please be aware, this is just a hobby for me, so I might be slow in replying.
